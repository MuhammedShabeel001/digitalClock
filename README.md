# Digital Clock

A beautiful, minimal, and elegant Digital Clock web application built with HTML, CSS, and Vanilla JavaScript.

## Features
- Clean, premium Apple-inspired interface
- Glassmorphism design elements
- 12/24 hour format toggle
- Light/Dark theme switch (with local storage persistence)
- Smooth, performant animations
- Fully responsive layout

## Folder Structure
- `css/` - Contains modular CSS files (variables, reset, layout, components, responsive).
- `js/` - Contains modular JS files (format helpers, theme toggler, clock logic, main app entry).

## How to Run
Since this project uses ES6 modules, you'll need to serve it via a local web server (e.g., Live Server extension in VS Code, `npx serve`, or `python3 -m http.server`).

Enjoy!
